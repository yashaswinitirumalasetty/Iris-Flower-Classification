# Iris Flower Classification

This project uses Machine Learning to classify Iris flower species using the K-Nearest Neighbors (KNN) algorithm.

## Technologies Used

* Python
* Scikit-Learn
* NumPy

## Features

* Loads the Iris dataset
* Splits data into training and testing sets
* Trains a KNN classifier
* Predicts flower species
* Evaluates model accuracy

## Result

The model achieved 100% accuracy on the test dataset.

## How to Run

```bash
py iris.py
```
